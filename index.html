Welcome to the Campus Resource Booking System

This project was developed for Concordia University's SOEN 287 course.

The system allows students and faculty to reserve university resources such as study rooms, labs, and sports facilities. Administrators can manage resources, bookings, and monitor usage.

Features for Students/Faculty

  • Create and manage accounts
  • Browse available resources
  • Make, modify, or cancel bookings
  • View upcoming and past reservations

Features for Administrators

  • Add, edit, and remove resources
  • Approve or reject booking requests
  • View usage statistics
